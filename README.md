# Granulat Financial Analysis

Analiza finansowa segmentu Granulat za lata 2023–2025.

## Zawartość
- analiza wyników finansowych
- rachunek zysków i strat
- bilans
- cash flow
- wskaźniki płynności, zadłużenia i rentowności
- analiza trendów i red flags
- wersja PDF do pobrania
